# Test Cases — Project 03 (Authentication)

## TC-01 — Successful sign in with valid credentials

### Preconditions
- User has a registered account

### Steps
1. Open the application in Incognito mode
2. Navigate to Sign in
3. Enter valid email and password
4. Click the "Sign in" button

### Expected Result
User should be authenticated and redirected to the Home page.

### Actual Result
FAIL — Sign in button gets stuck and authentication does not complete.
